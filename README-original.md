## 🚀 快速开始

### MVP阶段

```bash
# 克隆项目
cd projects/twinturing

# 安装依赖（待开发）
npm install

# 启动开发服务器
npm run dev

# 构建静态站
npm run build

# 部署到Vercel
vercel deploy
```

## 🛠️ 技术栈

### MVP阶段

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **部署**: Vercel
- **图片**: Cloudinary
- **分析**: Vercel Analytics

### 生产环境

- **后端**: Node.js + Express
- **数据库**: PostgreSQL + Prisma
- **缓存**: Redis
- **支付**: Stripe
- **部署**: Vercel + AWS
